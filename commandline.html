server¶

Runs the tracking server

observatory server [OPTIONS]

Options

--port <port>¶

The port the server should listen on for incoming tracking data. By default the server listens on port 5001. Please be aware that you need additional configuration on the client when you change the default port.

--es-node <es_node>¶

Pass in one or more ip-addresses/hostnames of elastic search nodes to connect to. The application will automatically try to discover more nodes, so in principle you need only set one node.
